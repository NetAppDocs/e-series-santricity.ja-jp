= Snapshotにはどのような種類のボリュームを使用できますか？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Snapshotイメージを格納できるボリュームは、標準ボリュームとシンボリュームだけです。標準以外のボリュームは使用できません。ベースボリュームはプールまたはボリュームグループのどちらかに配置できます。
