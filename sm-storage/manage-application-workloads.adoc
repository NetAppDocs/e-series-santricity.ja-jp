= アプリケーションとワークロードの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
アプリケーションとワークロードを管理して、ストレージアレイ上の既存のすべてのボリュームをワークロード名とアプリケーションタイプでグループ化して表示できます。また、既存または新規のワークロードにボリュームを追加したり、既存のワークロードの設定を変更したり、新しいボリュームとワークロードを作成したりすることもできます。
