= 非同期ミラーリング-ボリュームが一部表示されないのはなぜですか？
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ミラーペアのプライマリボリュームを選択すると、対応するすべてのボリュームのリストが表示されます。

使用できないボリュームはリストに表示されません。次のいずれかの理由で、ボリュームが対象外になっている可能性があります。

* 最適状態でない。
* すでにミラー関係に参加している。
* シンボリュームの場合は、自動拡張を有効にする必要があります。
