= SNMPまたはsyslogのアラートを設定するにはどうすればよいですか？
:allow-uri-read: 
:experimental: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Eメールアラートに加えて、アラートが簡易ネットワーク管理プロトコル（SNMP）トラップまたはsyslogメッセージで送信されるように設定できます。

SNMPまたはsyslogのアラートを設定するには、メニューの[アラート]に移動します。
